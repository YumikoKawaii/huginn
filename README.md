# Huginn

**Overview**

In Norse mythology, Huginn is one of Odin's two ravens — the raven of *thought* — sent out at dawn to fly across the world and return with everything worth knowing. This project does the same thing, but for manga.

Huginn is a self-hosted pipeline that autonomously hunts down manga chapters from across the internet, archives them, and keeps the shelves stocked without anyone having to ask. A scheduled crawler wakes up every hour to check what is new, and a traffic bot breathes life into the archive around the clock.

**What it does**

It knows which series matter. It checks what is already home. It fetches only what is missing, then delivers it. Everything runs on AWS, everything is containerised, and once deployed it requires no human attention whatsoever.
